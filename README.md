# Nodaz Frontend

Nodaz Frontend is a React, Redux Toolkit, React Router, and Tailwind storefront.

The app uses the Node backend for all data and auth flows.

## Configuration

Copy `.env.example` values into your local environment as needed.

Backend configuration:

```env
VITE_API_URL=http://localhost:8000/api
```

## Development

```bash
npm install
npm run dev
```

## Accounts

Create an account from the signup page before logging in, or use the backend-seeded admin account if the server created one.
